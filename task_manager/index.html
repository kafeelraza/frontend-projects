Attributes vs Properties

Type in the input and compare live property value with original/changed HTML attribute.

input.value

Initial Markup Value

input.getAttribute("value")

Initial Markup Value

Comment Explanation

Attribute: The value written in HTML markup. It is accessed with getAttribute() and changed with setAttribute().

Property: The live DOM object value. For inputs, input.value changes immediately while the user types.

Browser Rendering Pipeline

Displayed using cards, a flow diagram, timeline-like ordering, and boxes connected with arrows.

1. HTML

Browser receives HTML bytes and converts them into characters.

2. Parsing

Characters are parsed according to HTML rules.

3. Tokenization

HTML is converted into tokens such as start tags, end tags and text.

4. DOM Tree

Tokens become DOM nodes arranged as a tree.

5. CSSOM Tree

CSS is parsed into the CSS Object Model.

6. Render Tree

DOM and CSSOM combine to create the render tree for visible content.
