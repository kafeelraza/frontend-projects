# 🚀 Gaming Website Landing Page — Futuristic Gaming Studio UI

A premium futuristic gaming studio landing page built using **HTML5** and **CSS3** with modern UI/UX effects, glassmorphism, neon glow aesthetics, hover animations, and responsive design.

This project was created as a task from **Sheryians Coding School — Cohort 3**.
